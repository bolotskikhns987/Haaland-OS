Timing / по времени дня

Добавки не заменяют приём пищи и не должны ухудшать сон или аппетит.
До тренировки

Еда, вода, стимуляторы только по необходимости

Нормальный приём пищи за 1.5–3 часа. Кофеин 100–150 мг за 45–60 минут только при необходимости. Для ног/скорости опционально цитруллин 6–8 г. Не использовать кофеин вечером.

После

Сначала калории и белок

Еда 700–1200 ккал или гейнер, если обычная еда не лезет. Whey 25–35 г только если белок не добран. Креатин можно принять в любой удобный момент.

Вечер

Восстановление без стимуляторов

Магний 200–300 мг элементарного. Whey 25–35 г перед сном только при провале по белку. Кофеин и агрессивные предтрены запрещены.

День отдыха

Базу не отменять

Креатин 5 г оставить. Калории не резать. Омега-3 и витамин D — по текущему плану; коллаген возможен перед мобилити при показаниях.
